# .dotfiles

## Install

```
$ git clone https://github.com/michaelbragg/dotfiles.git ~/.dotfiles && cd ~/.dotfiles
$ chmod 755 scripts/bootstrap
$ scripts/bootstrap
```

### OSX

Run `osx/set-defaults.sh` and you'll be good to go.

### ZSH

Switch to using ZSH by default for Terminal:

```
chsh -s $(which zsh)
```
